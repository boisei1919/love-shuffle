/* --- [수정] 글쓰기 버튼 클릭 시: 모든 게시판 목록 생성 및 현재 게시판 자동 선택 --- */ document.querySelector('.write-btn').onclick = function() { // 1. 화면 전환 views.boardDetail.classList.add('hidden'); views.write.classList.remove('hidden'); // 2. 전체 게시판 목록 정의 (여기에 있는 것만 선택 가능) const allBoards = [ "자유게시판", "비밀게시판", "졸업생게시판", "새내기게시판", "장터게시판", "원룸/주거", "알바/과외", "분실물", "연애상담", "취업/진로", "홍보게시판", "동아리/학회", "대외활동", "총학생회", "방송국", "공과대학", "인문대학" ]; // 3. 선택창(Select) 비우고 다시 채우기 const selectEl = document.getElementById('write-board-select'); selectEl.innerHTML = ""; // 기존 옵션 초기화 allBoards.forEach(boardName => { const option = document.createElement('option'); option.value = boardName; option.innerText = boardName; selectEl.appendChild(option); }); // 4. 현재 보고 있던 게시판(currentBoardName)을 기본값으로 선택 if (currentBoardName && allBoards.includes(currentBoardName)) { selectEl.value = currentBoardName; } else { selectEl.value = "자유게시판"; // 기본값 } };
